### 商品属性

- メーカーの管理メニューから遷移可能な「商品属性」のページに、CSV入稿時に利用する「サブマスタ区分」と「サブマスタ区分値」の値が記載されておりますので、ご参照ください。

| カラム | 名称 | 型 | 長さ | PK | FK | 必須 | 例 | 備考 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| 1 | 型式（品番） | 文字列 | 255 | | | o |  |  |
| 2 | サブマスタ区分 | 文字列 | 255 | | | o | bath |  |
| 3 | サブマスタ区分値 | 文字列 | 255 | | | o | reheating |  |
